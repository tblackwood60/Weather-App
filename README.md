# Weather App

A small web application that displays weather conditions for any city. 
Built with HTML, CSS, and JavaScript, using a free public weather API.


## Features

- Current temperature and weather state

- Forecast on next 5 days

- City search

- Auto refresh


## Technologies

- HTML5

- CSS3

- JavaScript (ES6+)

- OpenWeatherMap REST API

- Animate css


## Installation

git clone https://github.com/tblackwood60/Weather-App
cd Weather-App
open index.html in browser
