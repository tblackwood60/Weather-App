# Weather App

A small web application that displays weather conditions for any city. 
Built with HTML, CSS, and JavaScript, using a free public weather API.


## Features

- Current temperature and weather state

- Multi-day forecast

- City search

- Auto refresh

- Light and dark themes


## Technologies

- HTML5

- CSS3

- JavaScript 

- Fetch API 

- Animate css


## Installation

git clone https://github.com/yourname/weather-app
cd Weather-App
open index.html in browser
